# Task Manager

**Website design to manage tasks and projects.**  
It also allows communication between members and generates reports.

---

## Technologies Used
- **React**
- **Node.js**
- **Express**
- **MongoDB**

---

## Installation

### Server
To install and run the server:

`cd backend`
`npm init -y`
`npm install cors dotenv express mongoose`
`npm start`

### Front End
To install and run the front end:

`npm install`
`npm start`
